# Gestion de Stock Agricole

Ce projet vise à créer une application de gestion des stocks pour une entreprise spécialisée dans l'agriculture et la production de plantes, d'arbres, de fleurs, etc. L'application permet de suivre les transferts, les produits, les ventes, les tableaux de bord, etc.

## Vue d'Ensemble

L'entreprise est composée d'un magasin principal (l'entrepôt) où le système est installé, ainsi que de trois centres de production et de vente. L'application aide à organiser les données d'inventaire et à maintenir un état juste des stocks, facilitant ainsi le travail du magasinier et permettant d'informer en cas de rupture de stock.

## Fonctionnalités Principales

- Enregistrement des arrivées de matières premières dans le stock depuis les fournisseurs.
- Transfert de portions de matières premières à chaque centre de production.
- Enregistrement des activités quotidiennes de chaque centre de production et de vente.
- Génération de rapports détaillés à partir des activités des centres.
- Gestion des ventes, y compris la possibilité de revendre des matières premières aux clients pertinents.
- Tableaux de bord pour une vue d'ensemble des stocks et des activités.

## Installation

1. Clonez le dépôt : `git clone https://github.com/votre-utilisateur/gestion-stock-agricole.git`
2. Accédez au répertoire : `cd gestion-stock-agricole`
3. Installez les dépendances : `pip install -r requirements.txt`
4. Configurez les paramètres nécessaires dans le fichier `settings.py`.

## Utilisation

1. Lancez le serveur de développement : `python manage.py runserver`
2. Accédez à l'interface d'administration : `http://localhost:8000/admin/`
